ADCS Educator Rig — Attitude Control Teaching Platform

Spacecraft Embedded Control Systems STEM Education

Hands-on satellite attitude control teaching platform with real IMU sensors, reaction wheel actuators, and a live quaternion attitude display — bridging classroom ADCS theory and physical hardware.

PROJ-003
Image 1 — Rig Hardware
Assembled ADCS educator rig with reaction wheels and IMU
Image 2 — Attitude Visualization
Live quaternion attitude display
Problem Statement

Attitude Determination and Control Systems (ADCS) are among the most conceptually challenging subsystems in spacecraft engineering. Without hands-on hardware, students struggle to internalize the relationship between sensor fusion, reference frames, control moment gyroscopes, and stability — concepts that are abstract when studied only through simulation or lecture.

A low-cost, physically interactive rig was needed to bring ADCS principles to life in an educational setting.

Solution & Implementation

Designed and built a tabletop educator rig replicating the core ADCS loop: IMU-based attitude determination (accelerometer + gyroscope fusion), attitude error computation in quaternion space, and closed-loop control output to reaction wheel motors. Students can physically disturb the rig and observe the control system respond, tune PID gains in real time, and visualize attitude on a live display.

The rig serves as a bridge between classroom theory and satellite hardware, reinforcing concepts taught in spacecraft systems courses.

Wireless DAQ & Control for Satellite Test Rig

Arduino Simulink Wireless DAQ

Tether-free data acquisition and closed-loop control for a freely rotating satellite test rig — Arduino wireless nodes stream sensor data to a Simulink real-time model that commands actuators back without any physical cable constraints.

PROJ-005
Image 1 — Wireless Node Hardware
Arduino wireless node mounted on satellite test rig
Image 2 — Simulink Block Diagram
Real-time control model in Simulink
Problem Statement

Satellite test rigs that physically articulate or rotate cannot use tethered DAQ cables without introducing unacceptable torque loads or cable wrap constraints on the rig's motion. Traditional wired data acquisition becomes a hardware bottleneck that limits the fidelity of testing for attitude control and dynamics experiments.

A wireless data link was needed to decouple the rig's mechanical freedom from its measurement and control infrastructure.

Solution & Implementation

Designed an Arduino-based wireless data acquisition system that streams sensor measurements — IMU, current, encoder — from the freely moving satellite rig to a ground-side host over a low-latency wireless link. The received data feeds directly into a Simulink real-time model that closes the control loop, commanding actuators back through the same wireless channel.

The architecture eliminates all tethers from the rig while preserving the real-time control loop timing required for ADCS experiments.

Supersonic tunnel DAQ front panel — live pressure traces

Supersonic test section during a run
Problem Statement

Supersonic wind tunnel experiments demand precise, synchronized measurement of multiple high-speed pressure and temperature channels during very short run windows. Missing data, unsynchronized channels, or delayed actuator commands during a run can invalidate an entire test, wasting facility time and experimental resources.

A robust, hardware-timed acquisition and control system was needed to guarantee data integrity and safe automated operation of the tunnel through its full run sequence.

Solution & Implementation

A state machine manages the full tunnel run sequence — pressurize, hold, fire, vent — with sensor-triggered transitions and hardware abort interlocks that protect the facility if any parameter exceeds safe limits.
